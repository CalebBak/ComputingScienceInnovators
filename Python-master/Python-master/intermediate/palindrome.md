# palindrome
The goal of this program is to check whether or not a word is a palindrome.  
## Sample input ## 
    >>Enter a word: hello
    >>hello is not a palindrome
    >>Enter a word: racecar
    >>racecar is a palindrome
   
## Reference ##

Gives you length of a string
